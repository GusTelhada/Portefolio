Tehates

Tehates is a software for Diy RC Transmitters. Its developed for ESP32 but works on other MCU with no problems. The UI is design in photoshop and functionality is programmed on the nextion app. Currently its only compatible with nextion displays (Uart Communication).

3DNS

3D Navigation System is a project that focus on spacial position for drones to use in autonomous missions. 5 diferent sensors are combined to achive this: GPS + Barometer + Accelerometer + Gyroscope + Magnatometer

Flight C.

The flight controller is the device that makes all the calculation necessary to make a drone fly. Currently I'm using the STM32f103c8t6, capable of a clock of 72MHz making 250 loops per second.

G.Station

The Ground Station is a Desktop app that is used to monitor drones on long range autonomous mission. Its programmed in visual studio 2022 in C#.
